Fitness teaching APP

Users just answers a few questions to advise their workout equipment. Not only teaching mode that provided general fitness teaching, but also game mode that users could finish how many repetitions in one minute and keep their workout moves correctly. Users even could do repetitions until muscle failure and achieve interesting and effective training. We develop user interface with Xcode. The teaching mode and the game mode use Mediapipe for image recognition, and compute the angle of the skeleton of each moves. If users reach the correct angle, it will be counted. Furthermore, there are prompt subtitles displayed for users in teaching mode. Therefore, the users could operate the teaching mode to familiarize themselves with the actions before playing the game mode.
